# small-kernel

## Description
1. Boot loader
2. Operating system kernel, including
    1. ISRs to handle CPU interrupts
    2. VGA driver
    3. Keyboard driver
    4. Shell (color function)


## Usage

### Boot

```
make run
```

### Debug

```
make debug
```
